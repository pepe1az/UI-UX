# CampusDesk MVP

CampusDesk — учебный прототип сервиса обращений студентов.

## Как открыть

1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. Переключайте роли: `Студент`, `Исполнитель`, `Админ`.
4. Создавайте заявки, меняйте статусы, добавляйте комментарии и оценки.

## Что реализовано

- создание обращения студентом;
- список обращений;
- фильтры по статусу и категории;
- поиск;
- карточка обращения;
- история действий;
- обработка заявки исполнителем;
- закрытие заявки;
- оценка закрытой заявки студентом;
- простая админ-панель;
- сохранение данных в localStorage браузера.

## Ограничения MVP

Это фронтенд-прототип без сервера и авторизации. Данные хранятся только в браузере.
